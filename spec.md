# Specification

## Summary
**Goal:** Add the PhoneStorm promotional image as a static asset and display it as a banner or featured card in the IdeaSpark Dashboard.

**Planned changes:**
- Place `phonestorm-banner.dim_1024x1024.png` in `frontend/public/assets/generated/`
- Add a banner or featured card component in the dashboard that renders the PhoneStorm image as a static asset
- Ensure the image is responsive and does not overflow its container
- Style the section to integrate with the existing dark background and card-based layout

**User-visible outcome:** Users see the PhoneStorm promotional banner displayed as a featured section within the dashboard, visually consistent with the dark/warm-accent theme.
